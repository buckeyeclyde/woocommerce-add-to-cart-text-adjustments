== Description ==
A WordPress plugin that adds an admin page where the WooCommerce Add to Cart button text can be changed.

== Installation ==
Copy the woocommerce-add-to-cart-text-adjustments folder and contents to /wp-content/plugins, and then activate the plugin from the WordPress admin dashboard.
